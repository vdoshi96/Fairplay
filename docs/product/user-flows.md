# User Flows

## Household Setup

1. Create a household.
2. Invite or add household members.
3. Choose initial responsibility categories.

## Responsibility Planning

1. Add or select a responsibility.
2. Define expected outcome, cadence, and current owner.
3. Add handoff notes or review timing.
4. Save to the shared planning board.

## Check-In

1. Review active responsibilities.
2. Mark what changed, stalled, or needs support.
3. Adjust owner, cadence, or status.
4. Record a summary for future reference.
