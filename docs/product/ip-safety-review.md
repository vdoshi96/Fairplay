# IP Safety Review

## Current Position

Private reference materials exist locally in `References/` and are excluded from git. They should not be copied, summarized at length, transformed into app content, or used as a source of proprietary structure without explicit clearance.

## Allowed for Setup

- Mentioning that private reference materials are excluded from git.
- Creating original planning documents based on the product goal.
- Recording open questions for later review.

## Requires Review

- Any extracted text, card names, spreadsheets, Trello export content, diagrams, or source-specific taxonomy.
- Any feature that closely mirrors a copyrighted source's expressive structure.
- Any import or migration workflow involving private materials.

## Future Process

Before implementation uses reference-derived ideas, document the source, proposed use, risk, and approval decision here.
