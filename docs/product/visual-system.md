# Visual System Direction

## Product Feel

Fairplay should feel calm, practical, and respectful. The interface should support repeated household planning conversations without making users feel judged or managed.

## Principles

- Use clear hierarchy and plain language.
- Prioritize scanability for responsibility lists and check-ins.
- Keep color purposeful for state, ownership, and urgency.
- Avoid playful or blame-oriented visual language.
- Favor accessible contrast and predictable controls.

## Early UI Needs

- Responsibility board.
- Assignment and handoff editor.
- Check-in review view.
- Household/member settings.
