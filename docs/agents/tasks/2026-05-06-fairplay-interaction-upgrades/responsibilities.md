# Responsibilities

## Controller

- Maintain the shared spec and implementation plan.
- Create isolated worktrees and branches for workstreams.
- Dispatch orchestrator agents with exact ownership boundaries.
- Review branch diffs before merging.
- Merge branches to `main` in the documented order.
- Run final full QA on merged `main`.

## Theme And Login Orchestrator

- Own `codex/fairplay-theme-login`.
- Implement system/light/dark theme mode and login Enter regression coverage.
- Update settings and theme docs.
- Provide focused test evidence before handoff.

## Integrated Qwen Art Orchestrator

- Own `codex/fairplay-integrated-qwen-art`.
- Update generation prompts and generated-art UI treatment.
- Ensure accepted AI card covers are visible on responsibility detail pages.
- Provide focused test and visual smoke evidence before handoff.

## Learn-By-Doing Guides Orchestrator

- Own `codex/fairplay-learn-by-doing-guides`.
- Expand guide practice into real page-level dummy workflows.
- Keep skip and exit behavior while blocking advancement until required work is complete.
- Provide component and Playwright guide evidence before handoff.

## Little Alex Physics Orchestrator

- Own `codex/fairplay-little-alex-physics`.
- Implement global decorative physics Little Alex with reduced-motion behavior.
- Provide component, app shell, and Playwright drag/fling evidence before handoff.

## Review Agents

- Run spec compliance review for each branch.
- Run code quality review for each branch.
- Run final merged regression review on `main`.
