# Responsibilities

## Controller

- Keep the spec, plan, and task docs current.
- Prepare isolated worktrees from the documented baseline.
- Dispatch orchestrator agents and review/fix agents.
- Merge feature branches in order.
- Run final full QA and GitHub SHA verification.

## Theme Dark QA Orchestrator

- Own `codex/fairplay-theme-dark-qa`.
- Implement the settings toggle and dark-mode readability fixes.
- Run screenshot-based dark-mode QA and focused tests.

## Greg Taskmaster Avatar Orchestrator

- Own `codex/fairplay-greg-taskmaster-avatar`.
- Add the static Greg avatar, correct Library button capitalization, remove the local Little Alex sidekick, and update guide copy/tests.

## Little Alex Upgrades Orchestrator

- Own `codex/fairplay-little-alex-upgrades`.
- Add persona-scoped Little Alex preferences, settings controls, and upgraded physics/idle/chat behavior.

## Review Agents

- Review each branch for requirement compliance.
- Review each branch for code quality and regression risk.
- Review the merged mainline before push.
