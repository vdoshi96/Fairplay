# Little Alex Coherent Assets

## Task

Investigate and implement the safest Little Alex asset strategy so future generated sprites describe one coherent original character with equal limb proportions and no copied internet/IP assets.

## Scope

- Inspect existing Qwen source sheets and sprite generation scripts.
- Prefer an original generated full-body/proportion template.
- Use a public-domain/open-license mannequin only if it is already local and its license can be documented.
- Focus on the asset contract, manifest prompt updates, and docs.
- Keep this work disjoint from e2e pixel QA.
