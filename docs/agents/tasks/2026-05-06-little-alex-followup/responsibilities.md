# Responsibilities

## Branch Owners

- `codex/little-alex-appearance-variants`: visible presentation differences while preserving suit and clipboard assets.
- `codex/little-alex-bounds-idle`: app-safe physics bounds, delayed stand-up, calmer idle standing, and turn-based randomized walking.
- `codex/little-alex-gaze-bubble`: cursor/touch gaze behavior and fling-only speech bubble behavior.

## Coordination

- Keep each branch focused on Little Alex behavior.
- Commit branch documentation with each branch.
- Review branch outputs before merging to `main`.
- Final integration owns conflict resolution, full QA, and GitHub SHA verification.
