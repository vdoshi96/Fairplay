# Learned

- Prisma `@default(uuid())` keeps `ResponsibilityTemplate.id` aligned with the UUID string id requirement while allowing seed data to use `slug` as the stable human-readable key.
- The initial migration generated by `prisma migrate diff --from-empty` is sufficient as a source-controlled deployment artifact for static review, but a live Postgres migration apply remains the stronger verification when a database is available.
- Repository integration coverage now directly expresses the scoping contract for the prior findings: cross-household writes reject related ids, and private radar drafts stay selected-persona scoped.
